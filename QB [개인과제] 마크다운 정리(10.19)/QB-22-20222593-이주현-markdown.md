# 마크다운 정리


## 헤더 (header)
  + #, ===, ---로 표현 가능

```
H1
===
H2
---
```
H1
===
H2
---
<br>

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```
# H1
## H2
### H3
#### H4
##### H5
###### H6


## 수평선 (hr)
  + *과 _로 표현 가능

```
* * *
***
*****
_ _ _
___
```

* * *
***
*****
_ _ _
___


## 목록 (list)
### 순서가 있는 목록
```
1. 하나
2. 둘
3. 셋
```
1. 하나
2. 둘
3. 셋
 
**내림차순으로 정렬**
```
1. 하나
2. 둘
4. 넷
3. 셋
```
1. 하나
2. 둘
3. 넷
4. 셋


### 순서가 없는 목록
  + *, +, -로 표현 
```
* 하나
  * 둘
    * 셋
+ 하나
  + 둘
    + 셋
- 하나
  - 둘
    - 셋
```
* 하나
  * 둘
    * 셋
+ 하나
  + 둘
    + 셋
- 하나
  - 둘
    - 셋
 
<br> **세 기호를 섞어서 사용 가능**
```
* 하나
 + 둘
   - 셋
```
* 하나
  + 둘
    - 셋


## 인용 (blockquote)
  + \>을 이용하여 표현 가능
```
> 가나
> > 다라
> > > 마바
```
> 가나
> > 다라
> > > 마바
> > > 


## 코드블럭
1. 줄 앞에 공백 4칸이 있는 경우

       코드블럭

2. `를 사용하는 경우
    (위 아래 3개씩)

 ```
 코드블럭
 ```
3. ~을 사용하는 경우
    (위 아래 3개씩)
~~~
코드블럭
~~~

\`을 사용하여 문장 사이 단어를 `블럭` 처리할 수 있음

## 테이블
  + 헤더와 셀을 구분할 때, 3개 이상의 - 기호가 필요
  + : 기호로 셀 내용 정렬 가능
```
헤더|헤더|헤더
---|---|---
셀|셀|셀
```
헤더|헤더|헤더
---|---|---
셀|셀|셀
```
헤더|헤더|헤더
---:|:---:|:---
셀|셀|셀
```
헤더|헤더|헤더
---:|:---:|:---
셀|셀|셀
